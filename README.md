Cuarto panel: campo promedio de transmitancia efectiva resultante de todo el periodo 
Quinto y sexto panel: mismo que el tercero y cuarto pero para PPFD
Séptimo y octavo: mismo que el tercero y cuarto pero para el tratamiento con mediasombra (24 de octubre - 10 de abril)
